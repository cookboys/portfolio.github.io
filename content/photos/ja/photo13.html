PostCSSでInstagram風の写真を作る

PostCSS {{< figure src="/images/photos/photo13_firstview.png" alt="">}}

元画像を用意する

今回はこの画像を使って、PostCSSで画像加工をしていきたいと思います。

プラグインをインストールする

      
npm install postcss-instagram
// or
yarn add postcss-instagram

// postcss.config.js
module.exports = (ctx) => ({
    require('postcss-instagram')
  ]
});

1977

      
/* PostCSS */
.postcss__instagram { filter: 1977;}

/* build */
.postcss__instagram {
  position: relative;
  display: inline-block;
}
.postcss__instagram img {
  filter: sepia(.5) hue-rotate(-35deg) saturate(1.6) contrast(.9);
}
.postcss__instagram:after {
  box-shadow: inset 0 0 3em #222;
  position: absolute;
  top: 0;
  right: 0;
  bottom: 0;
  left: 0;
  z-index: 1;
  content: "";
}
