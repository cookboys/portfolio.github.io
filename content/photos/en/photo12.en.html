Making letters of denim background and patchwork

Photoshop / Illustrator {{< figure src="/images/photos/photo12_firstview.png" alt="">}}

Create a texture of denim

Create a new layer and choose Layers → New Fill Layer → Pattern.

From the pattern menu, select "pattern 2" → "cloud pattern".

We will make the proportion "150%" and make a denim fill.

Duplicate the layer and change the drawing mode to "Overlay".

Select all the layers and copy / combine the layers with command / Ctrl + alt + Shift + E.

Apply "Layer" → "Smart Object" → "Convert to Smart Object" to the duplicated layer so that the original image does not break even if you add a filter.

Next, select "Filter" → "Sharp" → "Unsharp Mask" and apply with "Quantity: 50%", "Radius: 2 pixels", "Threshold: 1".

Adjust pattern

Set the drawing color to black (# 000000) and the background color to white (#ffffff).

Create a new layer and select "Filter" → "Drawing" → "Cloud pattern 1".

Select "Add Layer Mask" at the bottom of the Layers panel.

With the layer mask selected, execute the "cloud pattern 1" filter again.

Set the drawing mode to "Overlay" and adjust "Opacity".

Add patchwork border

Create a new layer.

Using "rectangular tool", create a frame path.

Open the Paths panel and save the path you created.

Open the "Brush" panel and set "Diameter: 10px", "Roundness ratio: 25%", "Spacing: 900%".

Check "Shape" and set "Jitter of angle: 0%" and "Control: Progress direction".

With the new layer created, open the "Path" panel again and select the saved path.

Select "Draw a path border with a brush" at the bottom of the "Path" panel.

Select "Layer Effect" and adjust "Bevel and Emboss" and "Drop Shadow" in appropriate amounts.

Add patchwork character

Open Illustrator and create a letter. For the font, I chose "Yuruka Std UB".

"Outlining" the text and adding a filter.

With text selected, select "Effect" → "Stylize" → "Graffiti".

Apply at "angle: 30 °", "displacement: 6 px", "line width: 4 px", "interval: 4 px", "displacement: 10 px" in order from the top.

Execute "Object" → "Appearance Split".

Execute "Object" → "Path" → "Path Outline".

Open the "Pathfinder" panel and select "Union".

Paste the created character into Photoshop. When pasting, paste with "smart object".

Select the pasted letters and adjust "Bevel and Emboss" and "Drop Shadow" of "Layer Effect" in appropriate amounts.

It is complete!!
