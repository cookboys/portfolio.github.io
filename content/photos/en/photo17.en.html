Prepare an image to edit with CSS3

In order to edit images with CSS3, we will prepare for the original image.

Usually I am accustomed to editing images using Photoshop or I can not figure out how much image editing can be done with CSS3, so I tried how far it can be processed.

Because it is a point, I decided to edit the image with raw CSS3Sass,Stylus,PostCSS

To make it easy to understand the change during processing, we chose an image that is as varied as possible and has no bias in the color balance.

Organize filter attributes

The elements used for image processing using CSS3 are mainly the filter attribute.

filter uses properties to edit images. We will first organize the properties.

  • blurblur element
  • grayscaleelement can be grayscale
  • contrastAdjust contrast
  • hue-rotateHue can be rotated
  • saturateAdjust saturation
  • brightnessYou can adjust brightness
  • invertIt is possible to invert gradation
  • sepiaCan sepia tone elements
  • drop-shadowYou can add a to the drop-shadow element
  • opacitypermeability can be adjusted

Also specifydisplay: block; position: relative;as the parent element.

By doing this, you can add more effects to the element by using the::afterattribute, which is one of the pseudo-classes (it is also possible to add shadows inside and overtones).

Blur

      
.filters__wrapper .blur__param--5 { filter: blur(5px);}
.filters__wrapper .blur__param--10 { filter: blur(10px);}

In Photoshop, it is equivalent to "filter" → "blur" → "blur (Gauss)".

Grayscale

      
.filters__wrapper .grayscale__param--50 { filter: grayscale(50%);}
.filters__wrapper .grayscale__param--100 { filter: grayscale(100%);}

In Photoshop, is it "image" → "color correction" → "lower saturation".

It is the same as "Image" → "Mode" → "Grayscale".

The point that the percentage can be adjusted is different.

Contrast

      
.filters__wrapper .contrast__param--150 { filter: contrast(150%);}
.filters__wrapper .contrast__param--300 { filter: contrast(300%);}
.filters__wrapper .contrast__param--50 { filter: contrast(50%);}

In Photoshop, it is equivalent to the contrast of "image" → "color correction" → "brightness / contrast".

Rotation of hue (hue-rotate)

      
.filters__wrapper .hue__param--90d { filter: hue-rotate(90deg);}
.filters__wrapper .hue__param--180d { filter: hue-rotate(180deg);}
.filters__wrapper .hue__param--270d { filter: hue-rotate(270deg);}

In Photoshop, it is equivalent to the hue of "image" → "color correction" → "hue / saturation".

What is different from "hue / saturation" is to change the hue from the rotation rate of the hue circle.

Hue circle is a round figure which visually expresses the hue of the three attributes of color proposed by Munsell "Hue (Hue), Saturation (Chroma), Lightness (Value)".

Neighboring color is "similar color" of the same line. The opposite color represents "complementary colors" of conflicting colors.

Saturation (saturate)

      
.filters__wrapper .saturate__param--150 { filter: saturate(150%);}
.filters__wrapper .saturate__param--300 { filter: saturate(300%);}
.filters__wrapper .saturate__param--50 { filter: saturate(50%);}

In Photoshop, it is equivalent to the saturation of "image" → "color correction" → "hue / saturation".

Brightness

      
.filters__wrapper .brightness__param--150 { filter: brightness(150%);}
.filters__wrapper .brightness__param--300 { filter: brightness(300%);}
.filters__wrapper .brightness__param--50 { filter: brightness(50%);}

In Photoshop, it is equivalent to the brightness of "image" → "color correction" → "hue / saturation".

Gradation inversion (invert)

      
.filters__wrapper .invert__param--60 { filter: invert(60%);}
.filters__wrapper .invert__param--100 { filter: invert(100%);}

In Photoshop, it is equivalent to "image" → "color correction" → "inversion of gradation".

The point that the percentage can be specified is different.

Sepia (sepia)

      
.filters__wrapper .sepia__param--50 { filter: sepia(50%);}
.filters__wrapper .sepia__param--100 { filter: sepia(100%);}

To realize with Photoshop, it is necessary to make sepia tone from "hue / saturation".

Shadow (drop-shadow)

      
.filters__wrapper .dropshadow__param--darker { filter: drop-shadow(4px 4px 8px #212121);}

In Photoshop, it is equivalent to "layer" → "layer style" → "drop shadow".

First, the shadow position in the horizontal direction (x axis).

Second, the shadow position in the vertical direction (y axis).

Third, the size of the shadow spread (blur).

Fourth, enter the color of the shadow. If you do not enter anything,black (# 000000)is applied.

Transparency (opacity)

      
.filters__wrapper .opacity__param--60 { filter: opacity(60%);}
.filters__wrapper .opacity__param--20 { filter: opacity(20%);}

In Photoshop, it is equivalent to "Opacity" in the layer menu.

It is the same effect as theopacityattribute.

Make frosted glass (applied technique)

      
.blurimage__wrapper,.blurimage__contents::before {
  background: url(/images/photos/photo17_01.png) no-repeat center;
  background-size: cover;
  box-sizing: border-box;
}
.blurimage__wrapper {
  position: relative;
  padding: 10%;
  margin: 32px auto;
}
.blurimage__contents::before {
  content: "";
  position: absolute;
  box-sizing: border-box;
  background-clip: content-box;
  width: 100%;
  height: 100%;
  padding: 10%;
  top: 0;
  left: 0;
  filter: blur(5px);
}
.blurimage__main {
  padding: 10%;
  position: relative;
  text-align: center;
  background: rgba(255,255,255,0.4);
  border-radius: 4px;
  box-shadow: 0 0 8px rgba(0,0,0,0.1);
}
.blurimage__main img {
  width: 30%;
}

It causes the same background image to be read in the::beforeattribute of the parent element and the child element.

Apply::beforeto thefilter: blur(5px);of the child element and apply only blurring to the background image of the child element.

box-sizing: border-box;Add attributes and let padding also be included in the size of the box.

Addbackground: rgba(255,255,255,0.4);to the grandchild element to make it look like white frosted glass.

In addition toposition: relative;specify it as parent element of relative position.
