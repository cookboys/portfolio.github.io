Make a colorful cube logo

Illustrator {{< figure src="/images/photos/photo11_firstview.png" alt="">}}

Create block

Use Illustrator "Rectangle tool" to create a square.

Select "Effect" → "3D" → "Extrude · Bevel".

Select "Position: Isometric method - left side" and change the "depth of extrusion" to an appropriate value (image is 130pt).

Duplicate the rectangle you created and change the color.

Select Object → Split Appearance.

Create parts

We will assemble the created blocks so that their parts are in close proximity.

Duplicate parts

After creating the parts, duplicate it to four.

Make a letter shape

After duplicating, delete the block so that it becomes each letter.

Finishing

After creating the letters, next we will make a shadow.

Create a rectangle and use "rotation tool" "shear tool" to make the rectangle laterally oblique.

In "Gradient Tool", paint the rectangle with a gradient of white to black to white.

Change "drawing mode" to multiplication. Adjust the opacity to an appropriate amount (image is 40%)

Select "Filter" → "Blur (Gauss)" and apply the blur with "Radius: 10 pixels".

Since the shadow was formed, I will place the logo slightly above.

In "Gradient Tool", create a background color.

It is complete! !
