Make a Instagram-like retro photo

Photoshop {{< figure src="/images/photos/photo08_firstview.png" alt="">}}

I will prepare photographs to be processed

I would like to process this picture and make an Instagram-like picture this time.

Adjust tone curve

Use "Layer" → "New Adjustment Layer" → "Tone Curve" and adjust as shown above.

Adjust level correction

Use "layer" → "new adjustment layer" → "level correction" and correct the black correction to "42".

Adjust color balance

Use "Layer" → "New Adjustment Layer" → "Color Balance" to correct colors.

Adjust the intermediate layer to "magenta: -10".

Adjust the shadow to "+25, -5, -10" in order from the top.

Adjust the highlight to "+12, -21, -25" in order from the top.

Adjust hue

Use "Layer" → "New Adjustment Layer" → "Hue / Saturation" to correct the color.

Check "Unify Hue".

The hue is set to "40", the saturation is set to "28", and the brightness is set to "-10".

Since the layer mask is refined, double click to adjust the layer mask.

I will fix it with the brush tool so that the soap bubbles and the person are reflected clearly vividly.

Adjust color

Select all layers and press command / Ctrl + Shift + Alt + E to duplicate the layers and combine all the layers.

Select the duplicated layer and execute "Image" -> "Color Tone Correction" -> "Reduce Saturation".

"Filder" → "blur" → "blur (Gauss)" at a radius of 20 px.

Change the drawing mode to "soft light". Leave the opacity at 100%.

Adjust the amount of light

Except for the layer created above, select all the layers again and press command / Ctrl + Shift + Alt + E to duplicate the layer and combine all the layers.

"Filder" → "blur" → "blur (Gauss)" at a radius of 20 px.

Change the drawing mode to "Screen".

Using the eraser tool, erase photos of people and soap bubbles so that the edges around the pictures remain.

Create a vignette effect

Create a new layer, change the drawing mode to "dodging (linear) - addition", and set the opacity to 30%.

Fill the layer with black (# 000000).

Select brush tool. Make it a very soft soft brush (hardness 0%).

Make the drawing color white (#ffffff) and paint the middle white, so that the edges around the photo remain.

Create a light leak

Create a new layer and select the brush tool again.

Make the drawing color soft red and draw red water drops.

Create a new layer and select the brush tool again.

Make the drawing color orange and draw a line with the brush tool from the middle of the circle.

Select "Filter" → "Blur" → "Blur (move)".

Set "Angle: 90 °" and adjust the distance.

Group layers

Select the layer of red circle and orange line and group it by command / Ctrl + G.

Change the drawing mode to "Screen".

Create a flare

In the same way, create a flare effect.

Create a pale pink and orange circle and group.

Change the drawing mode to "Screen".

It is complete!!
