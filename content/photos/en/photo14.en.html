Make a pattern overlapping delicate lines

Illustrator {{< figure src="/images/photos/photo14_firstview.png" alt="">}}

Create a background

Use "Rectangle tool" to create a background image of the gradation.

Using "pencil tool", I will make two wavy lines roughly.

Select "Object" → "Blend" → "Blend option" and set to "Step number: 60".

Select the two lines you created and select "Object" → "Blend" → "Create".

Create a line in the same way

In the same way, again create two lines with "pencil tool" and create a line using blend tool.

Change drawing mode

Select the line you created and change it from "Transparent" panel to "Drawing Mode: Overlay".

Set the opacity from the bottom line to "45%, 65%, 100%".

Create a character

Select "Text Tool", drag to create a text box, and fill in the text.

From the Paragraph pane, select Justify.

Enter the text and set the color to dark blue (# 000038).

Select the text box and let out the character.

Select Rectangle Tool and create a rectangle with the same dark blue line.

Once created, select text and rectangle and place it horizontally vertically centered from the Align panel.

Select the rectangle and choose Object → Path → Outline of Path and convert the rectangular line to fill.

Select text and rectangle again and make it a group.

Open the "Arrange" panel, change the setting to "Arrange on Artboard" and center it horizontally and vertically with respect to the artboard.

It is complete!!
