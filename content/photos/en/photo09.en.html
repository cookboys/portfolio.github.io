Brighten the picture of the night sky

Photoshop {{< figure src="/images/photos/photo09_firstview.png" alt="">}}

Prepare the image to be processed

In this time I would like to brighten the photos of the night sky using this image.

Apply CameraRaw filter

Before you modify it first, convert the image to "Smart Object". By doing this, it becomes possible to edit without destroying the original image.

Select "Filter" → "Camera Raw filter".

Set it to "Exposure amount: 1.25". This makes it possible to brighten the whole photo.

Open the "Tone curve" panel and adjust the contrast.

Return to the "Basic Correction" panel and adjust to "Color temperature: 16" and "Color cast compensation: 10" in "White balance" (values ​​are adjusted appropriately with the photos)

Open the "Detail" panel and set "Noise Reduction" to "Brightness: 45", "Luminance Detail: 35", "Brightness Contrast: 25". It can suppress the roughness of the star.

Open the "Lens Correction" panel and brighten the image using the "Exposure" slider.

When all settings are completed, press the "OK" button.

Adjust tone curve

Duplicate layers with command / Ctrl + J on the layer to which the filter is applied.

Use "Quick Selection Tool" to create selection ranges of mountain and forest pictures.

After creating the selection range, select "Layer" → "Layer mask" → "mask out of selection range".

With the original photo selected, use "Layer" → "New Adjustment Layer" → "Tone Curve" to create a new adjustment layer for the tone curve. As a result, it is possible to apply a tone curve only to the starry sky portion.

With the mask of the created adjustment layer selected, set the drawing color to black (# 000000) and fill it.

Continue to use the "brush tool" and paint the drawing color with white (#ffffff) at the place where you want to increase the sparkle.

Since the brush is hard in the painted area, it is blurred with "filter" → "blur" → "blur (Gauss)". Include a blur with "Radius: 250px".

Finish in pop

Make it even more pop. Select the original photo layer and duplicate again.

Select the duplicated layer, and select "Filter" → "Sharp" → "Unsharp Mask", then set "Amount: 88%" and "Radius: 10 px".

Further, "layer" -> "layer mask" -> "hide all areas" is set and a layer mask is set.

Select the brush tool, set the drawing color to white (#ffffff), and paint where you want to shine.

As before, in order to blur the brush, select "Filter" → "Blur" → "Blur (Gauss)" and set to "Radius: 250px".

Finally, adjust the fill or opacity of the layer to "50%".

It is complete!!
